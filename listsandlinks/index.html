How To Create a new Webpage on the Production Server

  1. Clone your repository and type cd green2350-cmis-wd.
  2. Type gedit index.html & and gedit main.css & to create the files
  3. Type python -m SimpleHTTPServer (in the repository folder) to run the webpage on the development server
  4. After you are done with your html and styling, check the webpage on the development server by going to localhost:8000/filename and validate your code. Then save your work and close it.
  5. In terminal, git add (filename), git commit -m"Desciption of what you did", and git push.
  6. To upload the file to the production server, start sftp by typing sftp green2350@cmis-hsteacher to connect to the production server.
  7. Type cd public_html and enter your password. Type !ls to view the files in the folder.
  8. Type put (filename) to upload the file or put * to put all the files.
  9. View the file by going to cmis-hsteacher/~green2350/filename
